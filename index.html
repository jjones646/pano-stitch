Introduction

This program takes in 2 images, and it stitches them together using feature detection.

       

After running the program with the provided example image, you will get a result like the one below.

The source code for this program is also hosted on GitHub at the following link:

https://github.com/jjones646/pano-stitch

The original objective was to implement a full 360 degree panorama view from some pictures I took on top of the CULC build a few months ago. You will notice that the source images for the google street view image at the following link are included. However, more advanced warping techniques are likely necessary to get the wide angle image to properly stitch together (I was unable to get successful results with OpenCV).

Workflow

  • Detect keypoints & extracts descriptors for both images using ORB
  • Match the descriptors between images using Brute Force matching
  • Estimate a homography matrix for lining up the images
  • Perform a warping transformation on the 2nd image
